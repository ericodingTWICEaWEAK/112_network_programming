## 請自行安裝pygame
在cmd輸入
pip install pygame
即可

## 啟動server
執行server.exe

## 開啟client
執行client.py

## 跑者操作：
按下 WASD、上下左右來移動

## 殺手操作：
按下 WASD、上下左右來移動
按下123控制陷阱(按下2代表2不會觸發陷阱，而1和3會觸發)
